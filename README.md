# node-react-redux-system

This system is useful for handling or managing a branches of bank. This system uses React and redux framework and node.js for the server. Also used bootstrap for styling.

## Installation
Install Crystal in your server. then
clone this repository
```
    git clone https://github.com/briza23/node-react-redux-system.git
```
Go to the repository:
```
    cd node-react-redux-system
```
Install yarn or npm:
```
    `yarn install` or `npm install`
```
Run the server:
```
    `yarn start` or `npm start`
```
the server will run at port `3000`

## Usage

A system that is used in a hackaton, that displays the details of the specific bank.
